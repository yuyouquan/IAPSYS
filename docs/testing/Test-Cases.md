## 优先级说明

| 优先级 | 含义 | 说明 |
|--------|------|------|
| P0 | 核心流程必须通过 | 阻塞发布的关键路径，回归必测 |
| P1 | 重要功能 | 功能完整性保障，每轮必测 |
| P2 | 次要功能/边界 | 边界值、异常、UI细节 |

## 附录

### A. 发布流程7节点说明

```
通道发布申请 → 通道发布审核 → 物料上传 → 物料审核 → 应用上架 → 业务内测 → 灰度监控
```

### B. 回退路径说明

| 回退触发节点 | 回退目标 | 触发条件 |
|-------------|---------|---------|
| 通道发布审核 | 通道发布申请 | 运营审核不通过 或 老板会签任一拒绝 |
| 物料审核 | 物料上传 | 物料审核不通过 |
| 业务内测 | 通道发布申请 | 基础信息错误 |
| 业务内测 | 物料上传 | 物料错误 |

### C. TypeSelector组件逻辑

| 左侧选择 | 右侧状态 | 说明 |
|----------|---------|------|
| 全部 | 清空并禁用 | 表示选择所有选项 |
| 包含 | 激活可多选 | 选中的为包含项，支持模糊搜索 |
| 不包含 | 激活可多选 | 选中的为排除项，支持模糊搜索 |

### D. 文件上传规格汇总

| 文件类型 | 格式 | 尺寸/比例 | 大小限制 | 数量 |
|---------|------|----------|---------|------|
| 应用图标 | jpg/png | 1:1，>=180x180 | - | 1张 |
| 置顶大图 | jpg/png | 1080x594 | <=2MB | 1张 |
| 截图 | jpg/png | 480x854 或 854x480 | <=2MB/张 | 3-5张 |

### E. 状态优先级计算规则

```
失败(红色) > 进行中(蓝色) > 已完成(绿色)
- 节点中有任一拒绝/失败 → 整体状态为失败(红色)
- 节点中有进行中且无失败 → 整体状态为进行中(蓝色)
- 所有节点全部完成 → 整体状态为已完成(绿色)
```
